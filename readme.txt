A native support forum for WordPress.

== Description ==

Using a third party support system (or e-mail, or none at all!) might not be the best approach for your project. Instead you may want to house all of your support tasks under the installation of WordPress that you're working on. The support section can be do anything, whether it be changes of colours to the site, or new content to be added - it's flexible. 

= Features =

* Adds a native support forum to WP Admin
* Dashboard statistics to show progress

= Roadmap =

* Ticket priorities
* Attach files/images to comments
* Record history of each ticket
* Improve ticket numbers (i.e. don't use WordPress post IDs)
* Ability to link tickets
* Ability to edit comments
* Notifications
* Select which users can view support
* Internationalization

== Installation ==

1. Upload the 'WP Support' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
